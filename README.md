# GDoc Converter (前端項目)

## 1. 項目概述
- **項目名稱**：GDoc Converter
- **項目描述**：一款Web應用工具，旨在幫助用戶將PDF文件高效、高保真地轉換為可協同編輯的Google Docs文件。
- **相關文檔**：
    - `PRD.md` (產品需求文檔)
    - `DESIGN_SPEC.md` (設計規範文檔)

## 2. 核心功能
- **Google 帳號登入**：使用 Google OAuth 實現一鍵登入，以便集成Google Drive。
- **PDF 文件上傳**：支持拖拽或點擊選擇本地PDF文件。
- **文件轉換**：(前端界面) 提供轉換進度顯示和轉換結果反饋。
- **Google Drive 集成**：(前端界面) 轉換成功後，提供「在Google Docs中打開」的直接連結。
- **轉換歷史**：(儀表板頁面) 展示用戶的歷史轉換記錄。

## 3. 技術棧
- **HTML5**：語義化網頁結構。
- **CSS3**：使用CSS變量實現設計規範中的色彩和佈局。
- **JavaScript (ES6+)**：實現頁面交互邏輯、API調用（模擬）和DOM操作。
